🔧 MyWeb 快捷操作手册
🚀 启动应用
python app.py → http://localhost:5000
管理员操作网址：http://localhost:5000/admin/dashboard 可以登录管理员网页,删除未验证用户

👑 管理员工具
python make_admin.py - 设置管理员权限
python debug_admin.py - 诊断管理员问题 




👥 用户管理
python showuser.py - 查看所有用户状态

📁 项目文档
python generate_docs.py - 生成结构文档PROJECT_STRUCTURE.md
python scan_project.py - 显示目录树

🧪 测试工具
python test_db.py - 测试数据库连接

🔗 重要访问地址
✅ 用户登录：/auth/login
✅ 用户注册：/auth/register
✅ 管理员后台：/admin/dashboard
✅ 用户仪表盘：/dashboard
✅ 退出登录：/auth/logout

⚠️ 调试地址
/dev/force-logout - 强制退出（开发用）
/list_routes - 查看所有路由（临时添加）

📊 文件统计
cat PROJECT_STRUCTURE.md - 查看项目结构
ls -la - 查看当前目录文件

📋 完整URL列表
页面	URL	说明
管理员登录	http://localhost:5000/auth/admin/login	👑 专属管理员登录
普通用户登录	http://localhost:5000/auth/login	👤 普通用户登录
用户注册	http://localhost:5000/auth/register	新用户注册
管理员后台	http://localhost:5000/admin/dashboard	管理员控制台
用户仪表盘	http://localhost:5000/dashboard	普通用户主页
